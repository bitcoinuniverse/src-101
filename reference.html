Validation checklist

Before broadcast.

Use this brief preflight every time a payload or transaction builder changes.

01

Exact version

Match the active spec, implementation version, and activation state.

02

Exact bytes

Confirm serialization, encoding, output placement, and field types before signing.

03

Exact state

Check ticker identity, available balance or supply, and indexer interpretation.

For production systems, independently reproduce validation from the maintained implementation and test against known transactions.
