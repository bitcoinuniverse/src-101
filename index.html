04 / Safety

Keep protocol state intentional.

Meta-protocol state is often stricter than the underlying Bitcoin transaction. A transaction can confirm while its intended protocol action is invalid.

  • SRC-101 is an evolving protocol. Do not copy a generic JSON example into a transaction and assume it is valid.
  • Verify activation height, carrier type, and current indexer release before integration.
  • Keep name assets separate from ordinary BTC UTXO workflows to avoid accidental handling or transfer errors.
